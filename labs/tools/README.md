# labs/tools/ — live domain tooling

If this course ships a **live domain visualizer** (a locally-served, single-file tool that streams the
learner's REAL environment state, with a per-module "lens" added over time), it lives here — one tool,
served by whatever mechanism the domain provides, adding one lens per module.

This is a per-course, optional deliverable. The pattern: a self-contained page served locally that reads
real state from the running lab environment and renders it in the same visual language as the course
simulators (see `../../SIM-TOOLKIT.md`). The sim teaches the model; the live tool shows reality.

Leave this directory empty (just this README) until the course defines its live tool.
